COMPUTER SCIENCE CLUB INSTALL PYGAME
Python launcher and IDE is already installed on school computers. If you are on your school laptop then you skip these two steps:
  1. Visit www.python.org and download the latest version of Python (3.13.0)
  2. Agree to Terms & Conditions and install launcher
Go to the Self Service app on your laptop, install Anaconda and Visual Studio Code. Open VSCode and go to the Extensions tab on the left. Install the Python extension.

python3 -m pip install -U pygame --user
Then hit enter. To check if that worked, run:
python3 -m pygame.examples.aliens
(OPTIONAL) If you get a warning to upgrade pip, run:
python3 -m pip install --upgrade pip
Pygame is installed on your computer! To start coding with it, try going to the Pygame tutorials at the CS Hub
