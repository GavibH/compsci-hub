COMPUTER SCIENCE CLUB INSTALL JAVA/ROBOCODE
Go to the Self Service app on your laptop and search "java". Then install "JDK: Java Dev Kit, via AdoptOpenJDK 11". This installed the Java Developer Kit (JDK) on your laptop so now your computer can handle Java!
Now let's install Robocode. Download the setup file and drag it to your Desktop (It might say "Unverified download blocked" but you can download it anyway)
Drag that setup file to your Desktop and open a new terminal window. In the terminal, enter:
cd ~/Desktop
Now that you're in the Desktop directory, enter:
java -jar robocode-a.b.c.d-setup.jar Where a, b, c, and d are the numbers in the file you downloaded.
Now Robocode is set up! You run robocode from the terminal, so go back to the terminal and type:
cd ../../../
If you see something like "~ f.lname26" in front of your cursor on the new terminal line, then you are in the right spot. Now enter to move into the correct directory:
cd robocode
And then to start Robocode:
sh ./robocode.sh
